À propos

Actuellement étudiant en deuxième année en DUT Informatique à l'IUT de Limoges, je vais faire un stage en Écosse, portant sur la création d'un jumeau numérique sémantique d'un bâtiment intelligent. Les technologies utilisées seront donc le Web Sémantique, le développement web et les API des capteurs.
